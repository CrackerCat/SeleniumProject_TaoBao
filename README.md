# SeleniumProject_TaoBao

## 项目简介

本项目是本人在学习 selenium 自动化测试时用所写的练手的项目，目前主要测试了淘宝的登录功能，由于找不到类似于秒杀的商品所以还没有完成关于清空购物车相关的功能。

## 项目组成

项目主要采用了 PO 设计模式

paeg : 该目录下主要存储被测试的页面中的元素

test_dir : 该目录下主要存储测试用例

test_report : 该目录下主要存储测试的结果

config.py : 该文件内容为相关的配置信息

run_test.py : 用来运行测试脚本

## 项目所使用到的技术

PO

Docker 

Selenium

Selenium Grid

Pytest

Jenkies
